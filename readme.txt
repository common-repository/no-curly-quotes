== Description ==

Stops WordPress turning inverted commas and double quotes ( ' and " ) into curly quotes in blog post content. Options to control behaviour of titles, post content, excerpt and comments separately: e.g. you can keep curly quotes in titles, but not in post content. Will work out of the box; no need to edit theme files.

== Installation ==

1. Upload `nocurlyquotes` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. It works automatically. Options can be set through Settings > No Curly Quotes.


== Frequently Asked Questions == 

= How do I go back to having curly quotes? =
Disable the plugin, or set all the options to `on` under Settings > No Curly Quotes. 

= I want curly quotes in post titles, but not in the post content. How do I do that? =
Change the options under Settings > No Curly Quotes. You can set behaviour separately for post titles, content, excerpt and comments. By default when you install the plugin, none of these will have curly quotes.
